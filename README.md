# EduHub — מאגר חומרים ראשית 🎓

אפליקציית React לניהול וגישה למאגר חומרי הכשרה של תוכנית ראשית.

## תכונות
- 477 קבצים מ-SharePoint עם כותרות ותיאורים
- חיפוש וסינון לפי נושא וסוג קובץ
- כניסת מנהל לעריכה
- הוספת קבצים חדשים

## הרצה מקומית
```bash
npm install
npm start
```

## פריסה ל-GitHub Pages
```bash
npm run deploy
```
